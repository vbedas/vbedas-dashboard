About This Project

This dashboard visualizes distributed architectural vibration configurations collected across an ESP-NOW point-to-point wireless array using MPU6050 Inertial Measurement Units. When high-frequency acceleration waves cross limits simultaneously within a 3-second correlation window, a dual-node confirmation flag triggers immediate telemetry serialization over regional servers and launches SMS alert queues.
