About This Project

This dashboard visualizes real-time vibration data collected by MPU6050 sensors connected to ESP32 microcontrollers. The system uses two nodes for verification. Node 1 updates primary display data, while Node 2 handles SMS alert operation.


The system does not predict earthquakes or calculate magnitude. It is designed as a low-cost local awareness prototype for abnormal ground vibration detection.
