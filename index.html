About This Project

This dashboard visualizes a dual-node vibration detection prototype. Node 1 handles the OLED display, buzzer, and LED alert, while Node 2 handles GSM/SMS notification. Both nodes exchange vibration status using ESP-NOW, and an alert is considered confirmed only when both nodes detect abnormal vibration within the allowed detection window.


The system does not predict earthquakes or calculate earthquake magnitude. It is designed as a low-cost local awareness prototype for abnormal ground vibration detection.
