About This Project

This dashboard visualizes real-time vibration data collected by an MPU6050 sensor connected to an ESP32 microcontroller. When the acceleration magnitude exceeds the predefined threshold, the system generates local alerts using a buzzer and LED, sends SMS notifications using a GSM module, and stores event data in Firebase.


The system does not predict earthquakes or calculate magnitude. It is designed as a low-cost local awareness prototype for abnormal ground vibration detection.
