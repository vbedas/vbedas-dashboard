About This Project

This dashboard visualises real-time vibration data collected by MPU6050 sensors connected to two ESP32 microcontrollers. Firebase /node1 is the Primary Display Node (OLED + Buzzer + LED). Firebase /node2 is the Secondary SMS Node (SIM800L).


The system does not predict earthquakes. It is a low-cost local awareness prototype for abnormal ground vibration detection using dual-node confirmation.
